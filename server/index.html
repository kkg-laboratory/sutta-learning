Documentary tool

Turn a seed of teaching into a sutta you can revisit.

This site is a quiet workshop: you offer a reflection or idea, and the system composes a Buddhist-style discourse, stores it, and lets you read it again like pages in a film—scene by scene, voice by voice.

Try now Open studio

Know-how

Built for documentary-minded practice: clear inputs, inspectable outputs, and a running log so you see what happened between thought and text.

I

Offer a source

Write a short seed—an image, a question, a line from life. You choose difficulty so the voice matches beginner warmth or master stillness.

II

Generate & store

The server asks the model for a title and a full discourse, then saves everything to the database so your suttas accumulate over time.

III

Browse & verify

Filter by difficulty, read the text alongside the original source, and use the console to audit requests and responses—useful when you are editing or teaching.

This is a creative and educational tool, not canonical scripture. Use discernment; treat outputs as material for reflection, not authority.
