# Sutta Learning

This project aims at creating new Suttas and revising existing ones. In the Buddhist culture, and Black culture suttas are a classical way to invoke the fire energy for consecrating new spaces, as well as maintaining old ones. With the programs in this project, we apply the "Suttas as a Program" (SaaP) methodology to uncover new dimensions, re-learn existing dimensions, update old dimensions, as well as heal or repair dimensions that are afflicted in this human life or Earth lifespan. With this project, we learn how to keep our consciousness in the right state and follow the Noble Eight Fold Path with clear comprehension, right mindfulness and right concentration.

# Instructions

1. Install Python 3.12.3 or the latest version of Python from the Internet via the official Python website.
2. Setup a virtual environment using the command `python -m venv .venv`.
3. Install the requirements using pip package manager: `pip install -r requirements.txt`.
4. Run the programs individually using the `python <file.py>` command.
